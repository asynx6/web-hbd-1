# 🎈 Web HBD Project

Sebuah proyek website ucapan ulang tahun interaktif yang dibangun menggunakan **React** dan **Vite**. Proyek ini dirancang dengan antarmuka yang modern, responsif, dan animasi yang halus.

## 🚀 Fitur Utama
- **Fast Performance**: Menggunakan Vite untuk bundling yang super cepat.
- **Responsive Design**: Tampilan optimal di berbagai perangkat (Mobile & Desktop).
- **Interactive UI**: Animasi menarik untuk memberikan kesan spesial.

## 🛠️ Tech Stack
- **Framework**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: CSS / Tailwind CSS (sesuaikan dengan yang kamu pakai)

## 📦 Cara Menjalankan Secara Lokal

1. Clone repository ini:
   ```bash
   git clone [https://github.com/asynx6/web-hbd-1.git](https://github.com/asynx6/web-hbd-1.git)```

1. Masuk ke folder project:
   ```bash
   cd web-hbd-1```

3. Install dependencies:
   ```bash
   npm install```

4. Jalankan server development:
   ```bash
   npm run dev```
